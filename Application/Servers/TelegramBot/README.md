# Telegram Bot

## Bot Functions

For administrators: Send commands and view data

For users: View data

### Commands

- Turn On Lights  
- Turn Off Lights  
- Add Water  
- Add Nutrients  

### Data options

- All recent data  
- Sensor 1 data  
- Sensor 2 data
